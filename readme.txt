== Description ==
A powerful product customization tool that allows users to design custom products (t-shirts, hoodies, caps, etc.) with drag-and-drop functionality.

Features:
* Upload images from device with drag & drop
* Add text with 21+ fonts, colors, bold/italic/underline, alignment, curved text
* Add art from 14 categories with 100+ SVG icons
* Change product colors (16 preset + custom color picker)
* Front/Back/Sleeve design views
* Names & Numbers tool with 2-step workflow
* Undo/Redo support
* Save designs to WordPress database
* Responsive design

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Create a new page and add shortcode [product_designer]
4. Visit the page to use the designer

== Shortcode ==
Use [product_designer] on any page or post.
